The 3 things that must be true (check in DevTools)

The element that holds the text and every ancestor down to .card has min-width: 0 (or overflow: hidden).
Add once, globally:

.card, .card-head, .card-name, .title-wrap, .name-wrap { min-width:0; }


Chips/chevron are flex/grid non-growing (flex:0 0 auto or separate row on mobile).

The name uses two-line clamp (-webkit-line-clamp:2; white-space:normal;).
